# 📁 HexView — вывод файлов в шестнадцатеричном формате на C++

Это учебный проект на C++ — утилита, которая выводит содержимое файла в виде hex-дампа, аналогично классической команде `hexdump`. Программа показывает шестнадцатеричные коды байтов и их текстовое представление в удобном для анализа формате.

## 🎯 Формат вывода

Каждая строка содержит информацию о блоке из 16 байт файла:

```
0000000000: 48 65 6C 6C 6F 20 57 6F | 72 6C 64 21 0A 00 00 00 Hello World!....
```

| Поле | Описание |
|------|----------|
| `0000000000` | Смещение от начала файла (10 шестнадцатеричных разрядов) |
| `:` | Разделитель |
| `48 65 6C 6C...` | Первые 8 байтов (шестнадцатеричные коды) |
| `\|` | Вертикальная черта-разделитель |
| `72 6C 64 21...` | Следующие 8 байтов |
| `Hello World!....` | Текстовое представление (управляющие символы выводятся как `.`) |

## 🚀 Использование

### Синтаксис

```
hexview [параметры] имя-файла
```

## Параметры

| Параметр | Описание |
|----------|----------|
| `-n <число>` | Количество выводимых байт |
| `-s <число>` | Смещение от начала файла в байтах |

## Примеры

# Вывод всего файла

```
./hexview file.bin
```

# Вывод первых 100 байт

```
./hexview -n 100 file.bin
```

# Вывод, начиная с 21-го байта

```
./hexview -s 20 file.bin
```

# Вывод 50 байт, начиная с 31-го

```
./hexview -n 50 -s 30 file.bin
```

## ⚙️ Особенности

- Работает в бинарном режиме
- Размер блока фиксирован — 16 байт
- Управляющие символы (код < 32) выводятся как `.`
- Не использует STL-контейнеры

## 🔧 Сборка

```
g++ -std=c++11 main.cpp -o hexview
```

## 📝 Требования

- Компилятор с поддержкой C++11
- Стандартная библиотека C++
